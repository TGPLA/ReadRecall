#### 🔧 技术要点

##### 数据库设计（如需后端支持）

对于需要持久化的功能（阅读进度、高亮记录），考虑新增以下表：

```sql
-- 阅读进度表
CREATE TABLE reading_progress (
  id CHAR(36) PRIMARY KEY,
  user_id CHAR(36) NOT NULL,
  book_id CHAR(36) NOT NULL,
  epubcfi TEXT NOT NULL,
  last_read_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (user_id) REFERENCES users(id),
  FOREIGN KEY (book_id) REFERENCES books(id),
  UNIQUE KEY (user_id, book_id)
);

-- 文本高亮表
CREATE TABLE highlights (
  id CHAR(36) PRIMARY KEY,
  user_id CHAR(36) NOT NULL,
  book_id CHAR(36) NOT NULL,
  cfi_range TEXT NOT NULL,
  text TEXT NOT NULL,
  color VARCHAR(20) DEFAULT 'yellow',
  note TEXT,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (user_id) REFERENCES users(id),
  FOREIGN KEY (book_id) REFERENCES books(id)
);
```

##### 前端实现要点

1. **阅读进度持久化**：
   - 使用 `use-local-storage-state` 或自定义 localStorage Hook
   - 按 `user_id + book_id` 作为 key 存储
   - 阅读器初始化时自动加载上次位置

2. **搜索功能**：
   - 使用 react-reader 内置的 `searchQuery` 和 `onSearchResults` props
   - 高亮显示搜索结果
   - 支持上一个/下一个结果导航

3. **主题切换**：
   - 使用 `rendition.themes.override()` 方法
   - 支持预设主题（日间、夜间、护眼）
   - 主题设置持久化到 localStorage
