
# 阅读回响 - 数据库结构文档

## 📊 数据库概览

- **数据库名**: `reading_reflection`
- **字符集**: `utf8mb4`
- **排序规则**: `utf8mb4_unicode_ci`
- **版本**: v3.0

---

## 🗂️ 核心表结构

### 1. users - 用户表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | CHAR(36) | PRIMARY KEY | 用户唯一标识 UUID |
| `email` | VARCHAR(255) | UNIQUE NOT NULL | 用户邮箱，用于登录 |
| `password_hash` | VARCHAR(255) | NOT NULL | 密码哈希值（bcrypt） |
| `nickname` | VARCHAR(100) | | 用户昵称 |
| `avatar_url` | VARCHAR(512) | | 头像 URL |
| `created_at` | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| `updated_at` | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP ON UPDATE | 更新时间 |

**索引**: `idx_users_email` (email)

---

### 2. books - 书籍表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | CHAR(36) | PRIMARY KEY | 书籍唯一标识 UUID |
| `user_id` | CHAR(36) | NOT NULL | 所属用户 ID |
| `title` | VARCHAR(255) | NOT NULL | 书名 |
| `author` | VARCHAR(255) | NOT NULL | 作者 |
| `cover_url` | VARCHAR(512) | | 封面图片 URL |
| `description` | TEXT | | 书籍简介 |
| `chapter_count` | INT | DEFAULT 0 | 章节数量 |
| `question_count` | INT | DEFAULT 0 | 题目总数 |
| `mastered_count` | INT | DEFAULT 0 | 已掌握题目数 |
| `created_at` | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| `updated_at` | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP ON UPDATE | 更新时间 |

**索引**: `idx_books_user_id`, `idx_books_created_at`  
**外键**: `user_id` → `users(id)` ON DELETE CASCADE

---

### 3. chapters - 章节表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | CHAR(36) | PRIMARY KEY | 章节唯一标识 UUID |
| `book_id` | CHAR(36) | NOT NULL | 所属书籍 ID |
| `user_id` | CHAR(36) | NOT NULL | 所属用户 ID（冗余字段） |
| `title` | VARCHAR(255) | NOT NULL | 章节标题 |
| `content` | TEXT | NOT NULL | 章节文本内容 |
| `order_index` | INT | DEFAULT 0 | 排序序号（从小到大） |
| `paragraph_count` | INT | DEFAULT 0 | 段落数量 |
| `question_count` | INT | DEFAULT 0 | 该章节的题目数量 |
| `mastered_count` | INT | DEFAULT 0 | 该章节已掌握题目数 |
| `created_at` | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| `updated_at` | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP ON UPDATE | 更新时间 |

**索引**: `idx_chapters_book_id`, `idx_chapters_user_id`, `idx_chapters_order`  
**外键**: `book_id` → `books(id)` ON DELETE CASCADE, `user_id` → `users(id)` ON DELETE CASCADE

---

### 4. paragraphs - 段落表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | CHAR(36) | PRIMARY KEY | 段落唯一标识 UUID |
| `chapter_id` | CHAR(36) | NOT NULL | 所属章节 ID |
| `user_id` | CHAR(36) | NOT NULL | 所属用户 ID（冗余字段） |
| `content` | TEXT | NOT NULL | 段落文本内容 |
| `order_index` | INT | DEFAULT 0 | 排序序号（从小到大） |
| `question_count` | INT | DEFAULT 0 | 该段落的题目数量 |
| `created_at` | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| `updated_at` | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP ON UPDATE | 更新时间 |

**索引**: `idx_paragraphs_chapter_id`, `idx_paragraphs_user_id`, `idx_paragraphs_order`  
**外键**: `chapter_id` → `chapters(id)` ON DELETE CASCADE, `user_id` → `users(id)` ON DELETE CASCADE

---

### 5. questions - 题目表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | CHAR(36) | PRIMARY KEY | 题目唯一标识 UUID |
| `user_id` | CHAR(36) | NOT NULL | 所属用户 ID |
| `book_id` | CHAR(36) | NOT NULL | 所属书籍 ID（冗余字段） |
| `chapter_id` | CHAR(36) | NOT NULL | 所属章节 ID |
| `paragraph_id` | CHAR(36) | | 所属段落 ID |
| `question` | TEXT | NOT NULL | 题目内容 |
| `question_type` | VARCHAR(50) | NOT NULL | 题目类型：名词解释/意图理解/生活应用 |
| `answer` | TEXT | NOT NULL | 答案解析 |
| `difficulty` | VARCHAR(50) | NOT NULL | 难度等级：简单/中等/困难 |
| `knowledge_point` | VARCHAR(255) | | 相关知识点 |
| `mastery_level` | VARCHAR(50) | DEFAULT '未掌握' | 掌握程度：未掌握/学习中/已掌握 |
| `practice_count` | INT | DEFAULT 0 | 练习次数 |
| `last_practiced_at` | TIMESTAMP | NULL | 最后练习时间 |
| `created_at` | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| `updated_at` | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP ON UPDATE | 更新时间 |

**索引**: `idx_questions_user_id`, `idx_questions_book_id`, `idx_questions_chapter_id`, `idx_questions_paragraph_id`, `idx_questions_created_at`, `idx_questions_mastery_level`  
**外键**: `user_id` → `users(id)` ON DELETE CASCADE, `book_id` → `books(id)` ON DELETE CASCADE, `chapter_id` → `chapters(id)` ON DELETE CASCADE, `paragraph_id` → `paragraphs(id)` ON DELETE CASCADE

---

### 7. practice_records - 练习记录表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | CHAR(36) | PRIMARY KEY | 记录唯一标识 UUID |
| `user_id` | CHAR(36) | NOT NULL | 所属用户 ID |

**外键**: `user_id` → `users(id)` ON DELETE CASCADE, `question_id` → `questions(id)` ON DELETE CASCADE

---

### 8. prompt_templates - 提示词模板表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | CHAR(36) | PRIMARY KEY | 模板唯一标识 UUID |
| `user_id` | CHAR(36) | | 所属用户 ID（NULL 表示系统预设模板） |
| `name` | VARCHAR(100) | NOT NULL | 模板名称 |
| `question_type` | VARCHAR(50) | NOT NULL | 题型：名词解释/意图理解/生活应用 |
| `content` | TEXT | NOT NULL | 提示词内容 |
| `is_default` | TINYINT(1) | DEFAULT 0 | 是否为默认模板：0-否，1-是 |
| `is_system` | TINYINT(1) | DEFAULT 0 | 是否为系统模板：0-用户创建，1-系统预设 |
| `created_at` | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| `updated_at` | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP ON UPDATE | 更新时间 |

**索引**: `idx_prompt_templates_user_id`, `idx_prompt_templates_question_type`  
**外键**: `user_id` → `users(id)` ON DELETE CASCADE

## 🔗 表关系图

```
users (1) ──┬── (N) books
             │
             ├── (1) user_settings
             │
             ├── (N) practice_records
             │
             └── (N) prompt_templates (系统模板 user_id=NULL)
                   
books (1) ── (N) chapters
                   
chapters (1) ── (N) paragraphs
               │
               └── (N) questions
                   
paragraphs (1) ── (N) questions
                   
questions (1) ── (N) practice_records
```

## 🧹 集成测试数据清理顺序

**重要**: 删除数据时必须遵循以下顺序，以避免外键约束错误：

```
1. practice_records (练习记录)
   ↓
2. questions (题目)
   ↓
3. paragraphs (段落)
   ↓
4. chapters (章节)
   ↓
5. books (书籍)
   ↓
6. user_settings (用户设置)
   ↓
7. prompt_templates (提示词模板，仅删除用户创建的)
   ↓
8. users (用户)
```

## 📝 字段特殊说明

### 冗余字段设计
- `chapters.user_id`: 冗余字段，用于快速查询用户的所有章节
- `questions.book_id`: 冗余字段，用于快速查询书籍的所有题目
- `questions.user_id`: 冗余字段，用于快速查询用户的所有题目

### 可空字段
- `questions.paragraph_id`: 题目可能不关联具体段落
- `prompt_templates.user_id`: 系统模板的 user_id 为 NULL
- `practice_records.is_correct`: 可能未评分

## 🐛 已知问题修复

### 2026-03-25 prompt_templates 外键约束修复

**问题描述**：删除用户时报错 `#1451 - Cannot delete or update a parent row: a foreign key constraint fails`

**根本原因**：`prompt_templates` 表的外键约束 `fk_prompt_templates_user` 的删除规则是 `NO ACTION` 而不是 `CASCADE`。

**修复方案**：
```sql
ALTER TABLE prompt_templates DROP FOREIGN KEY fk_prompt_templates_user;
ALTER TABLE prompt_templates
ADD CONSTRAINT fk_prompt_templates_user
FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE;
```

**影响**：现在删除用户时，该用户在 `prompt_templates` 表中的记录会自动删除。
