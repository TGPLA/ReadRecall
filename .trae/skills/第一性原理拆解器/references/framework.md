
# 第一性原理框架 - 详细方法论

## 埃隆·马斯克方法（物理思维）

"将事物归结为最基本的真理，然后从那里开始推理。"

### 电池案例
- **传统思维**："电池很贵（600美元/千瓦时），一直都是这样"
- **第一性原理**：电池是由什么制成的？碳、镍、铝、聚合物
- **原子级问题**：这些材料在伦敦金属交易所的价格是多少？
- **发现**：原材料 = 80美元/千瓦时
- **重建**：优化工程制造 → 特斯拉的成本突破

## 拆解阶梯

### 第1层：表面问题
"我们需要更好的家长参与度"

### 第2层：功能需求
"家长需要感受到与孩子一天的连接"

### 第3层：情感核心
"家长对不能在场感到内疚"

### 第4层：基本真理
"人类需要他们所爱的人平安的证明"

### 第5层：原子级解决方案
照片 + 时间戳 + 孩子的脸 = 平安的证明
其他一切都是装饰。

## 常见假设陷阱

### "行业标准"陷阱
"每个人都这样做" → 为什么每个人都这样做？
通常是：遗留问题，而非逻辑

### "客户说"陷阱
"客户要求X" → 他们试图解决什么**问题**？
通常是：他们在描述解决方案，而非需求

### "技术存在"陷阱
"我们应该使用[某技术]，因为它存在" → 这服务什么基本需求？
通常是：解决方案在寻找问题

### "竞争对手在做"陷阱
"竞争对手有这个功能" → 它真的解决了真正的问题吗？
通常是：功能膨胀，而非价值

## 穿透性问题

1. "如果我们今天从零开始，我们会这样构建吗？"
2. "一个完全的外行人会觉得这有什么奇怪的？"
3. "如果这贵10倍，我们还会做吗？"
4. "如果这是非法的，我们会怎么做？"
5. "一个孩子能理解的版本是什么？"

## 与其他技能的集成

- 使用**逆向思维**来验证："这个基本解决方案会如何失败？"
- 使用**二阶后果**来预测："实施后会带来什么后续影响？"
- 使用**预演分析**在构建前："想象这个失败了。为什么？"
